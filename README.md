# What Is This?

This repository proivdes a Dockerfile that just builds the latest
version of [re2c](http://re2c.org/) under RHEL8.

# Disclaimer

This repository and built binaries are publicly available, but these
are not official ones. Also, the binaries are not (code)signed.

# Packages

In [release](https://github.com/yoshimori-hj/rhel8-re2c-build/releases) page,
`re2c-${VERSION}-rhel8.tar.gz` contains the binaries.

- `re2c-${VERSION}-rhel8.tar.gz`: re2c application
- `re2c-${VERSION}-rhel8-debug.tar.gz`: Debug information
- `re2c-${VERSION}-rhel8-debugsource.tar.gz`: Source files for debugging
